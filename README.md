# fCCAuthorsApp
The freeCodeCamp News Author Page is a web application that displays a list of news authors fetched from a public JSON API. Users can view author names, avatars, short bios, and links to their full author pages. The app supports pagination with a "Load More Authors" button.

## Features
- Display author name, avatar, short bio, and link to full author page.
- Fetch author data dynamically from a JSON API.
- Load more authors incrementally with a button click.
- Responsive design for different screen sizes.
- Error handling if data fails to load.

## Technologies Used
- **HTML** – Page structure and content.
- **CSS** – Styling and layout.
- **JavaScript** – Fetch API, DOM manipulation, and pagination logic.

## Usage
1. Open `index.html` in a web browser.
2. Browse the authors displayed.
3. Click **Load More Authors** to view additional authors.
4. Click an author’s link to navigate to their full author page.
